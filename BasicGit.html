Basic Git Commands

1.Configure user details

git config --global user.name "your username"

git config --global user.email "your useremail"

git config --global user.password "your userpassword"

2.Help on git commands

git --help

3.Initialise a git repository

git init

4.Clone a git repository

git clone <repository-url>

5.Add files to staging

git add <file-name>

git add .

6.Commit Changes

git commit -m "Your commit message"

6.Push Changes

git push origin <branch-name>
